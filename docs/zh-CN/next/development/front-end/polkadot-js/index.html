Polkadot-JS

Polkadot-JS 项目是一系列围绕着 Polkadot 和 Substrate 衍生的工具、接口、和软件库。

Note:: While the project is named after "Polkadot", know that these tools, interfaces, and libraries are fully compatible with any Substrate based chain.

Polkadot-JS API

Polkadot-JS API 是一系列用于与 Polkadot 和 Substrate 的节点沟通的接口库。

此 API 让开发者可利用 Javascript 来查询节点状况及与 Polkadot 或 Substrate 开发的区块链进行互动。

文档 GitHub

开始

Polkadot-JS API 的开发者文档可在其网站中找到。 跟着 起始指南 一步步做，来学习如何安装并开始使用 Polkadot-JS API 。

Polkadot-JS 应用

Polkadot-JS 应用是一个用作与 Polkadot 或 Substrate 开发节点互动而扩展性高的用户界面。

它是一個已預制好的面向用户的应用程序，透過它可讓用戶訪問到 Substrate 區塊链內所有功能的。

訪問應用 GitHub

与本地节点连接

要连接你的 Polkadot-JS 应用到本地节点，您必须到 Settings 頁，并将 endpoint to connect to 設置为: Local Node (127.0.0.1:9944)。

注意： 如果你是通过加密 HTTPS 连接来访问 Polkadot-JS 应用，你的浏览器需支持可连接到没加密的 WebSocket 端口。 Google Chrome 支援这功能，但 Mozilla Firefox 不支援。

Polkadot-JS 扩展

Polkadot-JS 扩展是一个试验性应用来透过浏览器的扩展来管理账户，并用這些账户來签署外部函数。 它还提供了简单接口与本扩展兼容的应用互动。
