Getting Started

Добро пожаловать в Хаб для разработчиков Substrate.

Substrate - это фреймворк следующего поколения для блокчейнов и в этом качестве он имеет три отличительных черты:

  • Динамическая, самоопределяющаяся функция перехода состояний.
  • Функциональность лёгкого клиента с первого дня.
  • Прогрессивный алгоритм консенсуса с быстрым производством блоков и адаптивной, детерминированной финализацией.

Продукты

В экосистеме Substrate вы можете создавать блокчейны, смарт-контракты и даже пользовательские интерфейсы! Здесь содержится краткий обзор различных продуктов, которые состоят в этом пространстве:

  • Substrate: Фреймворк следующего поколения для разработки блокчейнов, построенный на основе WebAssembly, Libp2p и консенсуса GRANDPA.
  • ink!: eDSL, основанный на Rust, для построения смарт-контрактов WebAssembly, совместимых с модулем Substrate.
  • Polkadot-JS/API: Перспективные и RxJS API для взаимодействия с цепями, основанными на Substrate, через RPC-вызовы.
  • Polkadot-JS/Apps: Основной интерфейс для взаимодействия со всеми функциями узла Substrate. Работает на Polkadot-JS/API.
  • oo7-substrate: API на основе ReactJS для взаимодействия с цепями, основанными на Substrate, через RPC-вызовы. Также известна как библиотека "Bonds".
  • интерфейс Substrate: базовый интерфейс, основанный на ReactJS, создан с целью быстрого построения новых интерфейсов для цепочек Substrate. Работает на oo7-substrate.
